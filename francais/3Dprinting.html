C’est quoi l’impression 3D

L’impression 3d est un procédé de fabrication additif qui crée un objet en trois-dimensions à partir d’un model digital. Il y a de divers façon d’imprimer en 3d. Au Makerspace de l’université d’Ottawa, on utilise MDF(Modélisation par dépôt fondu) ce qui fonctionne en découpant le modèle en couches, puis l’imprime une couche au dessus de l’autre. Le matériel que nous utilisons est un type de plastique connu sous le nom de PLA(Acide Polylactique). Ce plastique est spécial grâce à la petite variation de température qui permet de fondre et refroidir le plastique, ce qui facilite son utilisation.

HComment faire une impression 3D au Makerspace?

Au Makerspace nous avons différentes marques d’imprimantes. Pendant le processus d’impression, tu rencontrera soit Ultimaker, Makerbot ou Dremel. Le processus est généralement le même.

  1. Créer/trouver un model 3D
  2. Sauvegarder ou télécharger le model en .stl
  3. Ouvrir le model dans un découpeur
  4. Envoyer le cide à l’imprimante 3D
  5. Start the print

1. Créer un model 3D

Il y a beaucoup de façon de créer ou trouver un model 3d. Si tu veut chercher a travers un librairie, Thingiverse ou Youmagine. Ces sites sont de bonne façon de t’inspirer toi même. Si tu est plus comme une personne qui fait tout soi même, il y a plusieur programs que tu peux essayer.

Si tu es un débutant, essaye Tinkercad. C’est une application de design 3d basé sur un navigateur qui est facile à apprendre. Pour plus d’information regarde ce guide utile. Si tu a besoins de quelque chose un peu plus avancer, va voir Fusion 360.

2. Sauvegarde ou télécharge le modèle en .stl

C’est quoi un fichier .stl? C'est un format de fichier stéréolithographique. Ce type de fichier utilise une toile de polygone pour décrire un objet 3d. C’est la façon la plus efficace et le type de fichier par défaut pour la majorité des logiciel d’impression 3d.

Dans Tinkercad, clique sur Exporté, une nouvelle fenêtre va apparaître puis choisi .STL

Dans Solidworks, clique sur File puis Save as. Une nouvelle fenêtre vas apparaitre. Choisi le fichier de type .stl.

3. Ouvrir le model dans un découpeur

Votre fichier .stl contient un ensemble de triangles en espace 3d. Si tu l'envoie à une imprimante 3d, celui si ne sera pas quoi faire. Un découpeur “découpe” l’objet 3d en couches puis génère du code de machine. Certaines imprimantes fonctionne mieux avec différent découpeur. Les découpeur doivent être téléchargé sur votre ordinateur. Par contre, tous nos ordinateurs on déja tous les logiciels pour chaqu’une de nos imprimantes.

4. Envoyer le code à l’imprimante

Le Ultimaker = Cura

  1. Ouvrir fichier dans Cura
  2. Choisir les paramètres pour l'impression que tu veux pour ton impression(pour plus d’information sur les paramètres des imprimantes et autre chose que tu peux faire dans le programme clique ici)
  3. Sauvegarde le fichier. Sois certain que tu sauvegarde sur la carte SD/li>

Makerbot Replicator = Makerbot desktop

  1. Ouvrir Makerbot desktop
  2. Appuis ajouté fichier
  3. Choisi la bonne imprimante. Nous avons le Replicator 2 et Replicator 2x. Pour faire cela, clique sur “device” in haut de menu.Puis, “Select type of device” ce qui peut être un des deux imprimantes mentionné auparavant.
  4. Envois ton fichier au dispositif. Il y a deux façon de faire. Soit avec une carte SD ou un câble USB
    1. Carte SD: Cliquez “Export Print File” Les Replicators fonctionne seulement avec des Cartes SD de 2GB(ou plus petit). Si tu essaye avec une carte plus grande, l'erreur “SD card read error” apparaîtra.
    2. USB Cable: Connecter votre imprimante à votre ordinateur en utilisant un câble usb. Vous devriez voir l’imprimante au bas du bureau de l’écran du Makerbot. Cliquez “Print”. Une fois appuyez, vous ne pouvez pas débrancher l’ordinateur de l’imprimante.

Dremel = Dremel Idea builder

  1. Charge ton fichier .stl en cliquant le bouton charger sur le menu sur le côté droit. Tu peux modifier ton objet (taille, position, orientation) en utilisant le bouton sur le côté gauche.
  2. Cliquez sur “Build”

  3. Choisis tes paramètres et tes méthodes d'imprimeries. Sois utilise une carte SD ou un câble USB

    • Carte SD: Tu sauvegarde la machine a une carte SD et tu la bouge à l'imprimante.

    • Cable USB: Tu utilise un câble usb pour transférer l’information à l’imprimante. Ce qui est bien avec les Dremels c’est que le fichier est téléchargé à la machine. Donc, tu peut débrancher le câble une fois que le fichier est transféré.

5. Commence l’impression

Commencer ton impression est très simple, il faut simplement sauvegarder le fichier à une carte SD ou connecter l’ordinateur avec un câble usb et appuyer imprimer ou construire.

Ultimaker

  1. Sauvegarde ton fichier a une carte SD. N’importe quel taille fonctionne.
  2. Déplacez vous vers l’imprimante et insérez la carte SD dans la poche à l’avant de l’imprimante
  3. Allume l’imprimante. Il y a un bouton on/off situé ou côté gauche de l’imprimante. C’est aussi un bon moment de faire sur que le filament est chargé dans l’imprimante.
  4. UEn utilisant la poignée. Selectionner imprimer en tournant la poignée Pour sélectionner appuyer sur la poignée. Cela vas vous amener à la liste des fichiers de la carte SD. Descend la liste jusqu'à ce que tu trouve ton fichier. Les fichiers les plus récent son situé au bas de la liste. Sélectionner votre fichier, cela commencera l’impression.

Dremel

Si tu utilise un cable usb. L’impression commence après que le bouton “build” est appuyer sur ton ordinateur. Suis les étapes ci dessous si tu utilise la carte SD.

  1. Place ta carte SD dans la poche de carte SD. Cette poche est situé dans le côté droit vers l’avant de l’imprimante.
  2. Si l’imprimante n’est pas déjà allumé. Allume la avec le bouton “on/off” situé sur le côté droit vers l’arrière de l’imprimante.
  3. En utilisant l’écran frontal. Appuis “Build”(la photo d’une carte SD) Sélectionner et Appuyer “Build”.
